The Why

University

A course of professor Sven Apel called
Software-Product-Line Engineering.

Former Projects

Different Apps with same code base for different user roles.

Often led to copy code or complex reusable components.

Runtime configurations drops performance.

Performance

Using runtime configuration to adapt the behaviour leads often to a lot of conditional statements.

Additionally it produces a lot of unnecessary code which won't be executed, but increases load times.
